戳一戳事件引擎，实现了戳一戳消息的接收。

目前仅支持如下环境：

- Windows x64
- QQ 9.9.2-16183 及以上

开发者近期没有精力将此引擎移植到其他环境。

只能识别戳一戳消息的来源聊天/来源群。群内触发的戳一戳消息无法得知发送者。可以用于触发随机文字/表情包等内容回复。
